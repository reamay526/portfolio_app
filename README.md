and Python.

The project is deployed and accessible online: 👉
https://reamay526-portfolio-app-home-veqy4o.streamlit.app/

## ▶️ How to Run Locally

1.  Install dependencies:

``` bash
pip install -r requirements.txt
```

2.  Run the app:

``` bash
streamlit run Home.py
```
